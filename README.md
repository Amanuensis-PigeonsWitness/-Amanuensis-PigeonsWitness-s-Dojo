## Amanuensis: Pigeons Witness の道場（*Dōjō*）

**SET デジタル道場**（*Dōjō*）へようこそ。
我々は、**交剣知愛**（*Kōken Chiai* / 剣を交え、もって愛を知る）の導きの下、オンライン上の言説を、**正念（しょうねん）の実践**たる「**稽古**」（*Keiko*）へと**変成**（*transmutieren*）させる。

**知的衝突**（*intellektuelle Kollision*）を是とせよ。勝利のため非（あら）ず、**相互研鑽**（*Mutual Refinement*）のためなれ。

**倫理の鍔**（*Ethische Tsuba*）を**堅持**せよ。**理念に挑み、人格を尊べ**。
